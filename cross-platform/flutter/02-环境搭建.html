    # JDK

    下载地址 (opens new window)

    需要先创建一个账户，再选择合适的环境版本

    # 安装

    下载完成后，双击安装包，开始安装（一直下一步就好，无需特色配置）

    # 验证

    安装完成后，重新打开命令行（win+r然后输入cmd回车），输入java -version

    如果看不到效果。需要将 JDK 的安装路径，绑定的环境变量中。然后，重新打开命令行验证。

    JAVA_HOME = C:\Program Files\Java\jdkxxxxx
