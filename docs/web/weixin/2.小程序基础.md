### 注册小程序账号

+ [注册地址](https://mp.weixin.qq.com/wxopen/waregister?action=step1)
+ 注册过微信公共号的邮箱，不能再注册微信小程序账号
+ 注册成功后，在后台，获取小程序的AppID
 + 设置 > 账号信息
